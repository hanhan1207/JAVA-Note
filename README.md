## 2020.06.27 
  - 堆（Heap):存放对象实例，所有对象实例及数组都要在堆上分配
  - 栈（Stack):指虚拟机栈，用于存储局部变量，例如基本数据类型、引用类型。方法执行完，自动释放。
  - 方法区（Method Area）：存储已被虚拟机加载的类信息、常量、静态变量、编译后的代码等。
